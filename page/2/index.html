Posted in: Metasploit, Tools.

אז מה זה Metasploit ואיך ניתן להתשתמש בו בביצוע בדיקות חדירות?

Metasploit הוא framework לפיתוח והרצה של exploits.
כל exploit ש-Metasploit מכיל כתוב בצורה מודולרית כך שניתן לשנות את יעד התקיפה, לבחור את הגרסה אותה אנחנו תוקפים וכן לשנות את ה-payload שנזריק בזמן התקיפה.
Metasploit מכיל בנוסף שלל כלים שמסייעים למפתחי exploits למשל encoders ועוד.

Extract and Decrypt GPP Passwords

Posted by mitm on 08/01/2015
Posted in: Password Cracking, Source Code, Tools.

החל מגרסת Windows Server 2008 מיקרוסופט השיקו יכולת חדשה ל-Active Directory בשם Group Policy Preferences – GPP המאפשרת יכולות מתקדמות יותר לעומת Group Policy Object – GPO הוותיקה.

אחת מהיכולות הללו היא להשתמש בסיסמאות ששמורת בתוך אובייקט ה-GPP לצורך איפוס סיסמה אוטומטי למשתמשים מקומיים, הרצת משימות מתוזמנות בעזרת משתמש בדומיין, מיפוי כונני רשת ועוד.

הסיסמאות הללו נשמרות בתוך קובץ ה-XML המכיל את הגדרות ה-Policy כאשר הן מוצפנות ע”י Aes 256 אבל לשמחתנו המפתח המשמש להצפנה אינו חסוי והוא מפורסם באתר מיקרוסופט

אז איך אפשר להשיג את הסיסמאות?
